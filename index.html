2025年の主要アップデート:

    推論能力の進化

    2025年の最大のトピックは、AIが「即答する」だけでなく「考えて（推論して）から答える」ようになったことです。 以下は、抽象的な難問解決能力を測るベンチマーク（ARC-AGI等）の推定比較です。

    GPT-5.2 / o1

    複雑な数学や科学の問題で、博士課程レベルの推論能力を発揮。思考時間を確保することで精度が劇的に向上。

    Claude Opus 4.5

    コーディングやニュアンスの理解において極めて高いスコア。人間らしい自然な対話能力も維持。

    Human Expert

    専門家平均。2025年のトップモデルは、特定のタスクにおいてついに人間を超え始めています。

    UIの革命: Canvas & Artifacts

    チャット欄で会話が終わる時代は終わりました。2025年のスタンダードは「画面分割」です。
    左側で指示を出し、右側で成果物（ドキュメント、コード、アプリ）を直接編集・プレビューします。

    Chat Input
    ブログ記事の構成案を作って。もっとカジュアルに修正して。
    右側のエディタで修正案を作成しました。特定の箇所を選択してコメントも可能です。
    Editor / Preview

    Canvasは「文書・コードの推敲」に特化しており、ワードプロセッサのように直接編集できます。
